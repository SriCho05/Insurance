# **App Name**: LeadMagnet

## Core Features:

- Product Display: Display insurance products with names, descriptions, and a 'View Details' button.
- Lead Capture Form: A product-specific form to collect customer details: Name, Age, Contact Number, and optionally Email and City. Form will implement validation.
- Submission Notification: Sends immediate notifications containing collected customer data upon form submission. Leverages a tool that assesses when it should prioritize a given lead and incorporate that assessment in its output.
- Database Storage: Store collected lead information in a database.
- Admin Dashboard: Admin area for viewing and downloading lead data, plus basic search and filtering.
- Responsive Design: Ensure the app is responsive and works well on different screen sizes.
- Marketing: Implement SEO and promotional support.

## Style Guidelines:

- Primary color: Deep sky blue (#3498db) for trustworthiness and professionalism.
- Background color: Light gray (#ecf0f1) for a clean and modern feel.
- Accent color: Emerald green (#2ecc71) to highlight calls to action and important information.
- Body and headline font: 'Inter' sans-serif, clean and readable for all content.
- Use flat, line-based icons to represent different insurance products. Consistency in style is key.
- Use a grid-based layout to showcase insurance products. Maintain adequate spacing between elements.
- Subtle hover effects on product cards to provide feedback. Smooth transitions between pages.
